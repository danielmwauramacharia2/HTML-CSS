# Lesson 1: HTML Basics

**Goal:** Learn how to create basic HTML elements (button, paragraph, link).

**Key Takeaways:**
- HTML = instructions for browsers.
- Tags define elements: `<tagname>` ... `</tagname>`
- Attributes modify behavior: `name="value"`.
- Extra spaces/newlines are ignored.
- Syntax must be correct or browser won’t display elements.

**Core Elements:**
- `<button>` – creates a button
- `<p>` – creates a paragraph
- `<a href="">` – creates a hyperlink

**Practice:** Build a simple webpage with one button, one paragraph, and a link.
